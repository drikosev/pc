Synopsis  
--------  
See also gmp, mpc, mpfr, dragonegg34. Java occupies enough space and should not be in Stage1 languages. Use NOJAVA=X to disable Java. Classes in Fortran are partially implemented, ie no finalizers. Other Fortran bugs include but not limited to PRs 44672, 54070, 57965, 64674, 64787, 64933, and 65428. It cannot recompile the java classes in both MacOS Sierra 10.12 and RHEL 7.3, thus this option has been deactivated. Also in Linux you might have to add to the .bashrc file in your home folder the line: export LD_LIBRARY_PATH=/usr/local/lib64
