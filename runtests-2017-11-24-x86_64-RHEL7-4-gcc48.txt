Synopsis  
--------  
See also gmp, mpc, mpfr, dragonegg34. Use NOJAVA=X to disable Java. Classes in Fortran are partially implemented, ie no finalizers. Other Fortran bugs include but not limited to PRs 47034, 44672, 54070, 57965, 64674, 64787, 64933, and 65428.In Linux, you have to manually install some build requirements, ie: binutils, glibc-static, zlib-devel, gettext, dejagnu, texinfo, glibc, libc.so.6, libunwind, glib2-devel. It might be easier if you try to build at first the source RPM that will inform you in more detail. Also in Linux you might have to add to the .bashrc file in your home folder the line: export LD_LIBRARY_PATH=/usr/local/lib64
