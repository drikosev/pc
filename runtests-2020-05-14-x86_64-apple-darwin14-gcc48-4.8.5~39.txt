Synopsis  
--------  
See also gmp, mpc, mpfr, dragonegg34. Use NOJAVA=X to disable Java. Many newer Fortran PRs have been filed in GCC Bugzilla, stale ones include, but not limited to, PRs 47034, 57965, 90948, and 65428. Whereas the PRs 61830 and 86481 report memory leaks. The PR 48097 is Java related, Darwin specific but suspended. This package cannot fully recompile the java classes in both MacOS Sierra 10.12 and RHEL 7.3. Thus, this option has been deactivated.
